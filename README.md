# Java Bank / ATM Management System

## Description
A console-based ATM application developed using Core Java and Object-Oriented Programming principles.

## Features
- Secure PIN authentication
- Balance enquiry
- Cash deposit and withdrawal
- PIN change functionality
- Transaction history

## Technologies Used
- Core Java
- OOP Concepts
- Java Collections (HashMap, ArrayList)

## How to Run
1. Clone or download the repository
2. Open the project in Eclipse or IntelliJ IDEA
3. Run Main.java
